HETAUDA SCHOOL OF MANAGEMENT

Introduction

The school management system is a web-based system which will use as a platform for interaction between student, teachers and parents. While the main objective of this project is to computerize the paperwork in the system and automate the work. The computerization is done so that the storage of all the details regarding students and teachers will be stored in the system which makes system centralized and the chance of duplication of any data is minimized. While by doing automation to the system will reduce the time for storing any data in the system.

Implementation

The development process for this project leveraged advanced CASE tools and technologies to ensure efficiency and accuracy. Visual Paradigm and ERD Plus were utilized to design Use Case Diagrams and ER Diagrams, providing a clear visualization of system processes and database structures. PhpStorm served as the Integrated Development Environment (IDE), streamlining the coding process with its powerful features. The backend development was implemented using PHP, while JavaScript enhanced frontend interactivity, and HTML and CSS were employed for designing and styling the user interface. For database management, MySQLi was used to ensure reliable and efficient data storage and retrieval. These tools and technologies collectively supported the seamless execution and implementation of the project.

Conclusion

This School Management System will manage the school by which students data can easily retrived and post. The system will incorporate all the features of this School Management system. It provides the tools for maintaining students and it count total number of students of every class. There is a database which is maintained by Admin in which all the information of students with stored.
